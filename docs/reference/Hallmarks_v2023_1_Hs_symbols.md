# A list with Hallmark gene sets (v2023.1)

A list with Human gene symbols from the MSigDB's Hallmark gene set
collection (Liberzon et al., 2015). These gene sets are used to
demonstrate the PathwaySpace's summit mapping pipeline, which assigns
summits to an image space.

## Usage

``` r
data(Hallmarks_v2023_1_Hs_symbols)
```

## Format

list

## Source

MSigDB database (v2023.1).

## Value

A list object.

## References

Liberzon et al. The Molecular Signatures Database (MSigDB) hallmark gene
set collection. Cell Systems 1(5):417-425, 2015 Doi:
10.1016/j.cels.2015.12.004

## Examples

``` r
data(Hallmarks_v2023_1_Hs_symbols)
```
