# COSMIC-CGC genes mapped to PathwaySpace images

A data frame listing 'GeneSymbol' and 'Entrez' IDs from the COSMIC-CGC
database (Sondka et al., 2020). These genes are used to demonstrate the
PathwaySpace's summit mapping pipeline, which assigns summits to an
image space.

## Usage

``` r
data(CGC_20211118)
```

## Format

data.frame

## Source

COSMIC-CGC database (release v95, tier 1 collection).

## Value

A data.frame object.

## References

Sondka et al. The COSMIC Cancer Gene Census: describing genetic
dysfunction across all human cancers. Nat Rev Cancer 18, 696-705, 2018.
Doi: 10.1038/s41568-018-0060-1.

## Examples

``` r
data(CGC_20211118)
```
